# Prompt Templates

## Basic Template
```
Role: [who you are]
Task: [what you want]
Context: [background info]
Format: [how to respond]
```

## Example
```
Role: Expert editor
Task: Review this blog post
Context: It's for a tech blog aimed at beginners
Format: Bullet points with suggestions
```
