# Prompt.dev - Master AI Prompting

Your AI prompt improvement course website.

## Structure
- `index.html` - Landing page
- `course/` - 6 course modules
- `templates/` - Prompt templates
- `exercises/` - Practice exercises
- `tips/` - Quick tips

## Deploy
1. Push to GitHub
2. Connect to Vercel/Netlify
3. Add custom domain: b3prompt.com

## Course Modules
1. Prompt Basics
2. Structure & Format
3. Specificity
4. Refinement
5. Advanced Tactics
6. Applications

## Development
Edit markdown files in `course/` folder.
HTML updates in `index.html`.
